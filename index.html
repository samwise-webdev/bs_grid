The Grid System


Sized Using Units .col-#

Every row in Bootstrap has 12 units of space to divide up. Every division within a row will be a column. Each column class will start with 'col' and can have additional parts (col-1, col-sm-1 etc) depending on what you want to do with the columns

Responsive Grid

The responsive grid classes (col-sm-#, col-md-#, col-lg-# etc) allow for the column to take up that many units above the specified breakpoint. The col-sm will apply to anything above 576px, the col-md will apply to anything above 768px and so on.
